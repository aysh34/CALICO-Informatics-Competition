# CALICO Informatics Competition
This repository is for my participation in the CALICO (Computer Algorithms and Logic in Competition) Informatics Competition, hosted by UC Berkeley. CALICO is a programming competition that tests algorithmic problem-solving skills, knowledge of data structures, and coding proficiency across various difficulty levels.
