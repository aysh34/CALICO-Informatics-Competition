# CALICO Informatics Competition 2025
This repository is for my participation in the CALICO (Computer Algorithms and Logic in Competition) Informatics Competition 2025, hosted by UC Berkeley. CALICO is a programming competition that tests algorithmic problem-solving skills, data structures knowledge, and coding proficiency across various difficulty levels.
